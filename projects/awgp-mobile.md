A production iOS and Android app for [All World Gayatri Pariwar](https://www.awgp.org/), bundling guided yagnas with synchronized audio narration, interactive guides for daily rituals, a Gayatri Mantra Japa counter, the VicharKranti books library with an in-app PDF reader, and audio/video media from awgp.org.

**Stack:** Expo SDK 54 + React Native + TypeScript on the client, Supabase (Postgres, Auth, Storage, Edge Functions) on the backend, with edge-function scrapers keeping the catalog in sync with awgp.org.
