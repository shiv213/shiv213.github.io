A live API exposing a cleaned dataset of Gujarati words with parts of speech, IPA pronunciations, definitions, and example sentences. Powers the [Gujarati Word of the Day](/projects/gujarati-wotd/) site and is the data layer for the rest of my Gujarati language tooling.

### [Live API docs](https://gujarati.shivvtrivedi.com/docs) · [GitHub](https://github.com/shiv213/GujaratiAPI)

Inspired by former UPenn Professor Babu Suthar's [*Gujarati–English Learner's Dictionary (Second draft)*](https://ccat.sas.upenn.edu/plc/gujarati/guj-engdictionary.pdf) — a Nirman Foundation Project, Department of South Asia Studies.
